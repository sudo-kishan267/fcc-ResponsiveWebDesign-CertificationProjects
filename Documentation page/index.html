Introduction to Python

Python is a high-level, interpreted programming language that emphasizes code readability and simplicity. Its clean syntax allows developers to express concepts in fewer lines of code than many other languages, making it particularly accessible for beginners.

Key Features

  • Easy to Learn: Python's syntax is simple and intuitive, making it an excellent choice for beginners.
  • Extensive Libraries: Offers a rich standard library and numerous third-party packages to extend functionality.
  • Versatile: Supports multiple programming paradigms, including procedural, object-oriented, and functional programming.
  • Cross-Platform: Runs on various operating systems like Windows, macOS, and Linux.

Prerequisites

Before diving into Python, you should have:

  • A basic understanding of programming concepts.
  • Familiarity with the command line interface.
Setting Up Python

Installation

  1. Download Python: Visit the Official Python website and download the latest version.
  2. Install: Follow the installation instructions for your operating system.
  3. Verify Installation: Open a terminal or command prompt and type:
python --version

IDEs and Text Editors

  • IDEs: PyCharm, Jupyter Notebook, Visual Studio Code
  • Text Editors: Sublime Text, Atom, Notepad++
Basics of Python

Hello, World!

To get started with Python, write your first program:

print("Hello, World!")

Lists and Dictionaries

List

A list is a mutable sequence of values:

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Outputs: apple

Dictionaries

A dictionary is a collection of key-value pairs:

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Outputs: apple
Control Structures

Conditional Statements

Use if, elif, and else for decision-making:

age = 20
if age < 18:
    print("Minor")
elif age < 65:
    print("Adult")
else:
    print("Senior")

Loops

Iterate over a sequence:

for fruit in fruits:
    print(fruit)

While Loop

Continue as long as a condition is true:

count = 0
while count < 5:
    print(count)
    count += 1
Functions

Defining Functions

Use the def keyword to define a function: 

                        def greet(name):
                            return f"Hello, {name}!"
                        
                        print(greet("Alice"))
Object-Oriented Programming

Classes and Objects

Define classes and create objects:

                            class Dog:
                                def __init__(self, name):
                                    self.name = name
                        
                                def bark(self):
                                    return f"{self.name} says Woof!"
                        
                            my_dog = Dog("Buddy")
                            print(my_dog.bark())
Working with Libraries

Importing Libraries

Use import to bring in libraries:

import math
print(math.sqrt(16))  # Outputs: 4.0

Popular Libraries

  • NumPy: For numerical operations.
  • Pandas: For data manipulation and analysis.
  • Matplotlib: For data visualization.
  • Requests: For making HTTP requests.
